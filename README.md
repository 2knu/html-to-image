# html-to-image

> A pure JavaScript library which can turn arbitrary DOM node into a vector (SVG) or raster (PNG or JPEG) images. fork from [dom-to-image](https://github.com/tsayen/dom-to-image).

## Install 

```
npm install --save html-to-image
```
